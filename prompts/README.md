# AI Prompts for Toygres Development

This folder contains useful AI prompts and context documents to help with Toygres development.

## Available Prompts

- **`project-context.md`** - High-level project overview and architecture for AI assistants
- **`implement-activity.md`** - Template prompt for implementing Duroxide activities
- **`implement-orchestration.md`** - Template prompt for implementing Duroxide orchestrations
- **`debugging-guide.md`** - Common issues and debugging strategies

## Usage

When working with an AI assistant on Toygres:

1. Share the relevant prompt file as context
2. Reference the specific files you're working on
3. Include any error messages or logs
